An NFS (v2) Server for Windows written in Allegro Common Lisp

nfs is an NFS server for Windows written in Allegro Common Lisp.  It was written by Ahmon Dancy of Franz Inc.  The reason it was written was our dissatisfaction with current products on the market, and the incredible problems we had configuring them on Windows 2000.  In fact, all told, we spent about the same amount of time doing administration on the product we bought as Ahmon did authoring the Lisp-based version--it took him about a week to get it up, and another week to make it faster than the other version we had been using.

Enjoy.
